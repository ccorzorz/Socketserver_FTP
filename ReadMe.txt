测试帐号:
    成员管理后台:admin/admin
    客户端测试帐号:cc/123  或者从管理后台添加成员也可

功能:
    1.cd 切换目录,只能在家目录操作
    2.rm 删除文件,如果文件是目录,空目录直接删除,非空目录无法删除
    3.put 上传文件,无法上传目录,上传时请加绝对路径,支持断点续传
    4.get 下载文件,支持断点续传,无法下载目录
    5.客户端可以输入help查看帮助
    6.mkdir 创建目录
    7.用户密码使用hashlib加密
    8.下载完成后使用MD5校验内容一致性
    9.下载上传有进度条显示
    10.管理员创建成员时,需要配置磁盘配额

注意事项:
    1.需安装prettytable
    2.使用了os.sep判断分割目录,在mac上开发的,但不确定能否在windows上正常运行
